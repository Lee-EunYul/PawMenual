# PAWMENUAL
오늘의 반려견 케어 결정을 빠르게 도와주는 맞춤형 **반려견 생활 관리 웹앱**입니다.
프로필 기반 식생활 가이드, 건강 상태 기록, 위치 기반 주변 동물병원 안내, 훈련 기록을 한곳에서 관리할 수 있도록 만든 프로젝트입니다.

## 웹페이지
[PAWMENUAL](https://thankful-sand-0954bbe00.7.azurestaticapps.net/)

위 링크를 클릭해서 접속하세요.

## 웹/앱 시연 영상
시연 영상은 준비 중입니다.

## 1) 프로젝트 한 줄 소개

- **무엇을 해결하나요?**: 반려견 식사/건강/훈련을 따로 기록하면서 관리가 분산되는 문제
- **어떤 방식으로 해결하나요?**: 프로필 기반 맞춤 안내 + 위치 기반 병원 조회 + 통합 기록 관리
- **프로젝트 성격**: 단일 프론트엔드(SPA) 기반 반려견 케어 서비스

## 2) 어떤 앱인가요?

- 웹에서 바로 사용하는 반려견 맞춤 케어 서비스
- 반려견 정보를 한 번 입력하면 식생활/건강/훈련 화면에 맞춤 정보가 연동되는 구조
- 로컬 저장소 기반으로 가볍게 기록/조회 가능하고, Azure Static Web Apps로 배포된 구성

## 3) 핵심 기능

### 3-1. 프로필 기반 맞춤 케어

- 강아지 이름, 나이, 몸무게, 주식 타입(건식/습식/혼합), 특이사항 입력
- 프로필 저장 후 홈/식생활/훈련 화면에서 공통으로 활용

### 3-2. 식생활 맞춤 가이드

- 상황(식욕 저하, 건식 거부 등) 선택 기반 급여 방향 안내
- 반려견 조건에 맞춘 추천 방향과 제품 후보 제시
- 식사 반응(좋아요/보통/거부) 기록 저장

### 3-3. 건강 상태 기록 + 주변 동물병원 조회

- 증상/체크 항목 선택 후 건강 요약 및 주의 단계 제공
- 현재 위치 기반으로 주변 동물병원 조회 및 거리순 정렬
- 조회 결과를 건강 기록과 함께 저장

### 3-4. 반응형 훈련 가이드

- 훈련 결과(성공/실패)에 따라 다음 액션 분기
- 실패 원인 선택 시 재시도 플랜 제시
- 훈련 이력 저장

### 3-5. 통합 성장 기록 조회

- 식사/건강/훈련 로그를 타임라인으로 통합 조회
- 카테고리/날짜 필터로 기록 탐색

## 4) 기술 스택

### Frontend

- React
- Vite
- JavaScript (ES Modules)
- React Router DOM

### Data / Storage

- Browser LocalStorage
- 공통 스토리지 유틸(`app/src/lib/storage.js`)로 로그/프로필 관리

### Deployment

- Azure Static Web Apps (Free)
- GitHub Actions 기반 CI/CD 자동 배포

## 5) 프로젝트 구조

```text
PawMenual/
├─ app/
│  ├─ src/
│  │  ├─ components/   # 공통 레이아웃/하단 네비
│  │  ├─ pages/        # 홈/프로필/식생활/건강/훈련/기록 페이지
│  │  ├─ lib/          # localStorage 유틸
│  │  └─ assets/       # 이미지/아이콘 자산
│  ├─ public/
│  └─ package.json
├─ agent.md            # 단계별 에이전트 지침서
└─ 기획서.md            # 프로젝트 기획 문서
```

## 6) 로컬 실행 방법

```bash
cd app
npm install
npm run dev
```
